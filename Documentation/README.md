## Contributing

When adding new documentation:

1. **Playbooks**: Step-by-step procedures for common tasks
2. **Technical Manuals**: Deep technical documentation
3. **Setup Guides**: Project setup and configuration
4. **System Documentation**: Architecture and system design
5. **Handoffs**: Team handoff documents
6. **Quick References**: Quick reference guides
7. **Archive**: Move old/deprecated docs here

## Maintenance

- Keep documentation up to date with code changes
- Archive outdated documentation rather than deleting
- Update this index when adding new documentation
- Follow naming conventions: `UPPERCASE_WITH_UNDERSCORES.md` for guides, `Title-Case.md` for manuals
