# Faction Indicators

This folder contains faction symbol/icon assets.

## Asset Requirements
- **Format**: PNG with transparency
- **Recommended Size**: 128x128 pixels (square) or appropriate icon size
- **Usage**: Display faction affiliation on cards

## Files to Add
- Icons for each faction in the game
- Neutral/no-faction indicator (if applicable)
- Dual-faction indicators (if cards can belong to multiple factions)

## Design Notes
- Clear, recognizable symbols
- Should be visible at small sizes
- Consistent style across all faction icons
- Consider both colored and monochrome versions
