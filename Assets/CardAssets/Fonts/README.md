# Fonts

This folder contains the main font files for the card game.

## Asset Requirements
- **Format**: TTF, OTF, or Unity-supported font format
- **Usage**: Card names, descriptions, stats, and UI text

## Files to Add
- Primary font (for card names and headers)
- Secondary font (for card descriptions and body text)
- Numbers font (if using special styling for stats)
- Any additional display fonts

## Font Guidelines
- **Readability**: Must be readable at various sizes
- **Style**: Should match the game's theme and aesthetic
- **Weights**: Include multiple weights if available (Regular, Bold, etc.)
- **License**: Ensure fonts are properly licensed for commercial use if applicable

## Recommended Font Styles
- Fantasy/medieval style for fantasy card games
- Sci-fi/futuristic for sci-fi themes
- Clean sans-serif for modern/minimalist games
- Decorative serif for elegant/historical themes
