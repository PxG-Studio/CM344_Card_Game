# Card Backgrounds

This folder contains background textures and patterns for cards.

## Asset Requirements
- **Format**: PNG or PSD
- **Recommended Size**: 512x768 pixels (2:3 aspect ratio)
- **Usage**: Background layer behind card art and text

## Files to Add
- Default card background
- Faction-specific backgrounds (if applicable)
- Rarity-specific backgrounds
- Texture overlays and patterns

## Design Notes
- Should complement but not overpower card content
- Consider color schemes that work well with text readability
- Can include subtle patterns or gradients
