# UI Components

This folder contains general UI assets related to card display and interaction.

## Asset Requirements
- **Format**: PNG with transparency (for icons and elements)
- **Various Sizes**: Depends on UI element

## Files to Add
- Button backgrounds and states (normal, hover, pressed, disabled)
- Panel backgrounds for card details
- Dividers and separators
- Health/mana orbs or indicators
- Cost indicators
- Stat backgrounds/badges
- Selection highlights or glows
- Tooltip backgrounds

## Design Notes
- Consistent visual style with card frames
- Ensure good contrast for text readability
- Include all interaction states where applicable
- Consider scalability for different screen sizes
