# Card Frames

This folder contains the card frame assets for the card game.

## Asset Requirements
- **Format**: PNG or PSD with transparency
- **Recommended Size**: 512x768 pixels (2:3 aspect ratio)
- **Usage**: Base frame/border for all card types

## Files to Add
- Card frame template (neutral/default)
- Variant frames for different card rarities (common, rare, epic, legendary)
- Any decorative frame elements

## Design Notes
- Ensure proper transparency for the card art area
- Leave space for:
  - Card name (top)
  - Card stats/numbers (corners)
  - Card description (bottom)
  - Faction indicator (top corner)
