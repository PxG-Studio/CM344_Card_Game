# Directional Arrows

This folder contains directional arrow assets for gameplay mechanics.

## Asset Requirements
- **Format**: PNG with transparency
- **Recommended Size**: 64x64 to 256x256 pixels (depending on usage)
- **Usage**: Indicate direction of attack, movement, or effect

## Files to Add
- Up arrow
- Down arrow
- Left arrow
- Right arrow
- Diagonal arrows (if needed)
- Multi-directional arrows
- Curved/rotational arrows (if applicable)

## Design Notes
- Clear direction indication
- Should match the game's art style
- Consider both normal and highlighted/selected states
- May need different sizes for different UI contexts
