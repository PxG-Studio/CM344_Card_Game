# Elemental Modifiers

This folder contains elemental type icons and indicators.

## Asset Requirements
- **Format**: PNG with transparency
- **Recommended Size**: 64x64 to 128x128 pixels
- **Usage**: Display elemental attributes or damage types

## Files to Add
- Fire element icon
- Water element icon
- Earth element icon
- Air/Wind element icon
- Light element icon
- Dark element icon
- Neutral element icon
- Any additional elemental types

## Design Notes
- Clear, distinct visual style for each element
- Should be recognizable at small sizes
- Consider color coding (red for fire, blue for water, etc.)
- May include both full-color and simplified versions
